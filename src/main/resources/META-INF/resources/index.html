Implementierung: Micronaut

Diese Anwendung realisiert die Workflowautomatisierung des gegebenen BPMN-Diagramms mit Hilfe von Camunda BPM und Spring Boot. Diese Anwendung dient als Basis des Vergleichs meiner Bachelorarbeit. Es soll verglichen werden, welche Vor- und Nachteile die Realisierung desselben Projekts in Micronaut und Quarkus mit sich bringen.

Diese Anwendung besitzt kein Frontend, weswegen sie durch die REST-API gesteuert wird. Für POST-Anfragen kann Postman verwendet werden. Mit dem senden eines Urlaubsantrags wird der Workflow gestartet. Der nachfolgende Task "Urlaubsantrag überprüfen" ist ein Humantask, welcher zuerst einem User assigned wird und anschließlich vom User bearbeitet wird. (Ablehnung oder Bestätitung des Urlaubantrags). Danach wird durch einen Servicetask eine automatische Meldung über den Ausgang des Urlaubsantrags ausgegeben. Dieses Workflowautomatisierung ist auf keinen Fall vollständig. Es soll lediglich zum Vergleich mit Micronaut und Quarkus dienen.

Rest-API:

  • POST /api/Holidayrequests/create

    • Erstellt ein Urlaubsantrag und startet den Workflow. "fullName", "vorgesetzter", "startDate", "endDate" und "status" müssen im JSON-Format im Body übergeben werden.

  • GET /api/Holidayrequests

    • Zeigt alle Urlaubsanträge an.

  • GET /api/Users

    • Zeigt alle angelegten User

  • GET /api/Users/{id}

    • Zeigt einen bestimmten User an.

  • POST/api/Users/create

    • Legt einen User an. "name" muss im Body im JSON Format übergeben werden.

  • GET /api/Holidayrequests/{id}

    • Zeigt einen speziellen Urlaubsantrag an

  • POST /api/Holidayrequests/{id}/assign

    • Assigned einen User einem bestimmten Task. "userid" und "name" müssen im Body im JSON Format übergeben werden.

  • POST /api/Holidayrequests/{id}/approve

    • Bestätigt einen bestimmten Urlaubsantrag."userid" und "name" müssen im Body im JSON Format übergeben werden.

  • POST /api/Holidayrequests/{id}/reject

    • Lehnt einen bestimmten Urlaubsantrag ab."userid" und "name" müssen im Body im JSON Format übergeben werden.
